# Iris Flower Classification Using Machine Learning

## Project Overview
This project predicts iris flower species using the K-Nearest Neighbors (KNN) algorithm.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Flowers Classified
- Setosa
- Versicolor
- Virginica

## Features
- Dataset visualization
- Model training
- Flower prediction
- Accuracy calculation
- Classification report
- Separate flower graphs
