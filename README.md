# Iris Flower Classification Using Machine Learning

## Project Overview
This project is a Machine Learning classification project that predicts the species of iris flowers using the K-Nearest Neighbors (KNN) algorithm.

The model classifies flowers into:
- Setosa
- Versicolor
- Virginica

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook / Google Colab

---

## Machine Learning Algorithm
- K-Nearest Neighbors (KNN)

---

## Dataset
The Iris dataset is loaded from Scikit-learn datasets.

Dataset contains:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## Project Steps
1. Import Libraries
2. Load Iris Dataset
3. Create DataFrame
4. Split Training and Testing Data
5. Train KNN Model
6. Predict Flower Species
7. Calculate Accuracy
8. Generate Classification Report
9. Display Flower Graphs

---

## Model Performance
The project displays:
- Accuracy
- Precision
- Recall
- F1-Score
